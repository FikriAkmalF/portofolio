Skills

These are some of my skills

HTML

HTML adalah bahasa markup yang umumnya digunakan untuk menyusun bagian paragraf, heading, maupun link pada halaman web. Meskipun susunannya mirip dengan kode, perlu dicatat bahwa HTML bukanlah bahasa pemrograman. Keahlian saya dalam HTML memungkinkan saya untuk membangun dasar-dasar situs web dan mengonversi desain visual menjadi kode.

CSS

CSS adalah singkatan dari Cascading Style Sheets, yaitu bahasa yang digunakan untuk menentukan tampilan dan format halaman website. Dengan CSS, saya bisa mengatur jenis font, warna tulisan, dan latar belakang halaman. CSS digunakan bersama dengan bahasa markup, seperti HTML dan XML, untuk membangun sebuah website yang menarik dan memiliki fungsi yang berjalan baik. Keahlian saya dalam CSS memungkinkan saya untuk membuat tampilan website yang baik dengan cepat.

JavaScript

JavaScript adalah bahasa pemrograman yang digunakan untuk mengatur perilaku halaman web. Ketika sebuah halaman web lebih dari sekadar menampilkan informasi statis, JavaScript memungkinkan kita mengimplementasikan fitur-fitur kompleks seperti pembaruan konten secara berkala, peta interaktif, grafik 2D/3D animasi, dan pemutaran video.

Bootstrap

Bootstrap adalah salah satu framework paling populer di dunia untuk membangun situs web yang responsif dan mobile-first. Bootstrap memudahkan pengembang dalam merancang tata letak, mengatur komponen, dan mengintegrasikan JavaScript dengan cepat. Dengan Bootstrap, Anda dapat membangun situs yang terlihat bagus di berbagai perangkat, mulai dari desktop hingga ponsel.

Framework

Framework adalah kerangka kerja yang digunakan untuk membantu developer mengembangkan aplikasi atau website secara konsisten dan terstruktur. Framework memudahkan proses pengembangan dengan menyediakan struktur dasar, pustaka, dan aturan-aturan yang dapat digunakan untuk membangun aplikasi. Dengan menggunakan framework, developer dapat menghemat waktu dan mengurangi kesalahan dalam mengembangkan perangkat lunak.

Vue JS

Vue JS adalah kerangka kerja JavaScript yang digunakan untuk membangun antarmuka pengguna pada web. Berbeda dengan kerangka kerja lainnya, Vue JS memanfaatkan HTML, CSS, dan JavaScript standar, serta menyediakan model pemrograman deklaratif berbasis komponen. Dengan Vue js, developer dapat mengembangkan antarmuka pengguna secara efisien, baik untuk aplikasi sederhana maupun kompleks.
